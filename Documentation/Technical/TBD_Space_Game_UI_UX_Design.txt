1.0 UI/UX Design Philosophy
==========================
The TBD Space Game UI/UX design philosophy balances realism with usability to ensure complex simulation data remains accessible and engaging. Our guiding principles:

1.1 Design Principles
------------------
- **Clarity Over Complexity**: Present information clearly without overwhelming the player
- **Progressive Disclosure**: Reveal details at appropriate depths based on player expertise and current context
- **Consistent Interaction Patterns**: Maintain consistency across different UI modules
- **Diegetic Integration**: Immerse UI elements within the game world where appropriate
- **Accessibility First**: Design for universal access across diverse player abilities
- **Responsive Layout**: Adapt to various screen sizes and device capabilities
- **Performance Conscious**: Optimize UI for minimal performance impact

1.2 Player-Centric Design
----------------------
The UI/UX design centers on three player profiles:

- **Casual Explorers**: New players who focus on narrative and high-level management
- **Strategic Managers**: Mid-level players who focus on resource optimization and crew management
- **Technical Specialists**: Advanced players who engage with detailed simulation systems

Each interface component addresses the needs of all three profiles through layers of complexity.

2.0 Information Architecture
==========================
The information architecture organizes and prioritizes the complex simulation data into a coherent, navigable structure.

2.1 Information Hierarchy
----------------------
The game information is structured in three tiers:

1. **Critical Information** (Always Visible):
   - Current ship/habitat status
   - Immediate hazards or alerts
   - Active mission objectives
   - Resource critical thresholds

2. **Tactical Information** (One-Click Access):

   - Resource management
   - System status overviews

3. **Detailed Information** (Drill-Down Access):
   - Complete crew profiles
   - Detailed system diagnostics
   - Scientific data analysis
   - Historical trends and logs

4.0 Progressive Disclosure System
================================
The progressive disclosure system gradually reveals complexity based on player needs and experience.

4.1 Disclosure Levels
------------------
Information is presented in multiple layers:

1. **Essential**: Core information for basic gameplay (always visible)
2. **Standard**: Information needed for routine decisions (hover/single-click)
3. **Advanced**: Detailed data for optimization (nested menus)
4. **Expert**: Complete simulation details (specialized views)

4.2 Contextual Tutorials
---------------------
The interface introduces new elements progressively:

- **First Encounter**: Simplified version with guided interaction
- **Repeated Usage**: More features and options become visible
- **Mastery Detection**: Full complexity unlocked when player demonstrates competence

4.3 Dynamic Complexity
-------------------
The system adjusts interface complexity based on:

```csharp
/// <summary>
/// Manages progressive disclosure of interface complexity
/// </summary>
public class ProgressiveDisclosureManager
{
    private Dictionary<string, int> _featureInteractionCount = new Dictionary<string, int>();
    private Dictionary<string, float> _featureCompetencyScore = new Dictionary<string, float>();
    
    public void RecordInteraction(string featureId, InteractionResult result)
    {
        // Track how often player uses this feature
        if (!_featureInteractionCount.ContainsKey(featureId))
        {
            _featureInteractionCount[featureId] = 0;
        }
        _featureInteractionCount[featureId]++;
        
        // Update competency score based on successful interactions
        if (!_featureCompetencyScore.ContainsKey(featureId))
        {
            _featureCompetencyScore[featureId] = 0f;
        }
        
        float resultScore = EvaluateInteractionSuccess(result);
        _featureCompetencyScore[featureId] = Mathf.Lerp(
            _featureCompetencyScore[featureId],
            resultScore,
            0.2f  // Learning rate
        );
        
        // Adjust disclosure level based on competency
        UpdateFeatureDisclosureLevel(featureId);
    }
    
    private void UpdateFeatureDisclosureLevel(string featureId)
    {
        DisclosureLevel newLevel = CalculateDisclosureLevel(
            _featureInteractionCount[featureId],
            _featureCompetencyScore[featureId]
        );
        
        // Update the UI component with new disclosure level
        UIRegistry.GetComponentById(featureId)?.SetDisclosureLevel(newLevel);
    }

5.4 View Switching System
---------------------
The game provides seamless transitions between 2D and 3D representations of game elements, each optimized for different interaction patterns and information needs.

### View Modes
- **3D Spatial View**: Immersive representation of ship/habitat with realistic spatial relationships
- **2D Schematic View**: Simplified, information-dense representation optimized for management and analysis
- **Hybrid Overlays**: Contextual 2D information overlaid on 3D views for specific interactions

### Automatic View Transitions
The system automatically transitions between views based on zoom level:

```csharp
/// <summary>
/// Manages view transitions based on camera zoom level
/// </summary>
public class ViewTransitionManager : MonoBehaviour
{
    [SerializeField] private Camera _mainCamera;
    [SerializeField] private float _3Dto2DTransitionThreshold = 150f;
    [SerializeField] private float _2Dto3DTransitionThreshold = 120f;
    [SerializeField] private float _transitionDuration = 0.6f;
    
    private ViewMode _currentViewMode = ViewMode.Spatial3D;
    private bool _isTransitioning = false;
    
    public void Update()
    {
        if (_isTransitioning)
            return;
            
        float currentZoomLevel = _mainCamera.orthographicSize;
        
        // Check for 3D to 2D transition
        if (_currentViewMode == ViewMode.Spatial3D && currentZoomLevel > _3Dto2DTransitionThreshold)
        {
            TransitionToView(ViewMode.Schematic2D);
        }
        // Check for 2D to 3D transition
        else if (_currentViewMode == ViewMode.Schematic2D && currentZoomLevel < _2Dto3DTransitionThreshold)
        {
            TransitionToView(ViewMode.Spatial3D);
        }
    }
    
    /// <summary>
    /// Handle mouse wheel input for zooming
    /// </summary>
    public void OnMouseWheel(float delta)
    {
        float zoomSpeed = _currentViewMode == ViewMode.Spatial3D ? 
            _3DZoomSpeed : _2DZoomSpeed;
            
        float newZoomLevel = _mainCamera.orthographicSize - delta * zoomSpeed;
        
        // Clamp to valid zoom ranges
        newZoomLevel = Mathf.Clamp(newZoomLevel, _minZoom, _maxZoom);
        
        // Apply zoom with smooth damping
        _mainCamera.orthographicSize = Mathf.SmoothDamp(
            _mainCamera.orthographicSize, 
            newZoomLevel, 
            ref _zoomVelocity, 
            _zoomSmoothTime
        );
    }
    
    /// <summary>
    /// Transitions between view modes with smooth animation
    /// </summary>
    public void TransitionToView(ViewMode targetMode, bool immediate = false)
    {
        if (_currentViewMode == targetMode || _isTransitioning)
            return;
            
        _isTransitioning = true;
        
        // Save focus point to maintain context
        Vector3 focusPoint = GetCurrentFocusPoint();
        
        // Begin transition animation
        StartCoroutine(AnimateViewTransition(_currentViewMode, targetMode, focusPoint, immediate ? 0f : _transitionDuration));
        
        // Update current mode
        _currentViewMode = targetMode;
        
        // Notify systems about view change
        OnViewModeChanged?.Invoke(_currentViewMode);
    }
    
    // Implementation details...
}
```

```csharp
/// <summary>
/// Controls the quick view switch button

public class ViewSwitchButton : MonoBehaviour
{

        ViewMode targetMode = _viewManager.CurrentViewMode == ViewMode.Spatial3D ?

        _viewManager.TransitionToView(targetMode);

            { "method", "quick_switch_button" }
        });

    private void UpdateButtonAppearance(ViewMode currentMode)
    {
        // Show the opposite mode icon (what you'll switch to)
        _buttonIcon.sprite = currentMode == ViewMode.Spatial3D ? _2DViewIcon : _3DViewIcon;
        
        // Update tooltip
        _tooltipText = currentMode == ViewMode.Spatial3D ? 
            "Switch to Schematic View" : "Switch to 3D View";
    }

/// <summary>
/// Manages exterior camera that intelligently tracks objects of interest
/// </summary>
public class ExteriorCamera : MonoBehaviour
{
    [SerializeField] private Transform _cameraMount;
    [SerializeField] private float _rotationSpeed = 0.5f;
    [SerializeField] private Renderer _displayScreen;
    [SerializeField] private RenderTexture _cameraFeed;
    
    private SpaceObject _currentTarget;
    private Camera _camera;
    private CameraFeedData _currentFeedData;
    
    private void Awake()
    {
        _camera = GetComponentInChildren<Camera>();
        _camera.targetTexture = _cameraFeed;
        _displayScreen.material.SetTexture("_MainTex", _cameraFeed);
    }
    
    /// <summary>
    /// Sets camera to track a specific space object
    /// </summary>
    public void SetTarget(SpaceObject target)
    {
        _currentTarget = target;
        
        // Update camera feed data
        _currentFeedData = new CameraFeedData
        {
            TargetName = target.Name,
            TargetType = target.ObjectType,
            Distance = Vector3.Distance(transform.position, target.transform.position),
            RelativeVelocity = target.Velocity - ShipController.CurrentVelocity
        };
        
        // Update associated ship display
        ShipDisplay display = GetComponentInParent<ShipDisplay>();
        if (display != null)
        {
            display.UpdateDisplay(_currentFeedData);
        }
        
        // Log target acquisition
        Debug.Log($"Exterior camera tracking: {target.Name}");
    }
    
    private void Update()
    {
        if (_currentTarget != null)
        {
            // Smoothly rotate camera to track target
            Vector3 targetDirection = _currentTarget.transform.position - _cameraMount.position;
            Quaternion targetRotation = Quaternion.LookRotation(targetDirection);
            _cameraMount.rotation = Quaternion.Slerp(_cameraMount.rotation, targetRotation, _rotationSpeed * Time.deltaTime);
            
            // Update distance data
            _currentFeedData.Distance = Vector3.Distance(transform.position, _currentTarget.transform.position);
            
            // Add visual effects based on target type (stars get lens flare, etc.)
            UpdateVisualEffects();
        }
    }
    
    private void UpdateVisualEffects()
    {
        // Apply appropriate visual effects based on target type
        switch (_currentTarget.ObjectType)
        {
            case SpaceObjectType.Star:
                ApplyStarEffects();
                break;
                
            case SpaceObjectType.Planet:
                ApplyPlanetEffects();
                break;
                
            case SpaceObjectType.Ship:
                ApplyShipEffects();
                break;
        }
    }
}

### Focus Preservation
When switching between views, the system maintains focus on the selected element:

```csharp
/// <summary>
/// Maintains focus consistency between different view modes
/// </summary>
private Vector3 GetCurrentFocusPoint()
{
    // If an element is selected, use its position
    if (SelectionManager.HasSelection)
    {
        return SelectionManager.CurrentSelection.WorldPosition;
    }
    
    // Otherwise use camera focal point
    return _mainCamera.transform.position + _mainCamera.transform.forward * _focalDistance;
}

/// <summary>
/// Transitions between views while maintaining focus on the same element
/// </summary>
private IEnumerator AnimateViewTransition(ViewMode fromMode, ViewMode toMode, Vector3 focusPoint, float duration)
{
    // Disable input during transition
    InputManager.SetInputEnabled(false);
    
    // Setup start and end states for camera and UI
    CameraConfiguration startConfig = GetCameraConfig(fromMode);
    CameraConfiguration endConfig = GetCameraConfig(toMode);
    
    float startTime = Time.time;
    float progress = 0f;
    
    // Transition view indicator
    _viewIndicator.ShowTransitionState();
    
    while (progress < 1f)
    {
        progress = (Time.time - startTime) / duration;
        float easedProgress = EasingFunction.EaseInOutCubic(progress);
        
        // Interpolate camera position and properties
        _mainCamera.transform.position = Vector3.Lerp(
            startConfig.Position, 
            endConfig.Position, 
            easedProgress
        );
        
        _mainCamera.orthographicSize = Mathf.Lerp(
            startConfig.OrthographicSize,
            endConfig.OrthographicSize,
            easedProgress
        );
        
        // Interpolate UI element positions and scales
        foreach (UITransformable element in _transformableElements)
        {
            element.TransitionToView(fromMode, toMode, easedProgress);
        }
        
        yield return null;
    }
    
    // Apply final view mode-specific settings
    ApplyViewModeSettings(toMode);
    
    // Update view indicator
    _viewIndicator.ShowCurrentState(toMode);
    
    // Re-enable input
    InputManager.SetInputEnabled(true);
    
    _isTransitioning = false;
}

5.5 Context-Aware View Transitions
---------------------------------
The system intelligently adjusts camera views based on character movement and interactions in zero gravity:

### Zero-G Navigation Through Entryways
When a character approaches or interacts with doorways, the system intelligently manages view transitions:

```csharp
/// <summary>
/// Manages navigation and view transitions for doorways and interactable objects
/// </summary>
public class ZeroGNavigationManager : MonoBehaviour
{
    [SerializeField] private float _transitionDuration = 0.8f;
    [SerializeField] private float _doorwayApproachDistance = 1.5f;
    [SerializeField] private LayerMask _interactableLayer;
    
    private Character _activeCharacter;
    private ViewTransitionManager _viewManager;
    private Transform _camera;
    
    private void Awake()
    {
        _viewManager = GetComponent<ViewTransitionManager>();
        _camera = Camera.main.transform;
    }
    
    /// <summary>
    /// Handles user clicking on a doorway or interactable object
    /// </summary>
    public void OnInteractableObjectSelected(InteractableObject target)
    {
        // Get current active character (player or selected crew member)
        _activeCharacter = CharacterManager.GetActiveCharacter();
        
        // Calculate optimal path to interactable object
        Vector3[] path = PathfindingSystem.CalculateZeroGPath(
            _activeCharacter.transform.position, 
            target.GetInteractionPosition()
        );
        
        // Begin character movement
        StartCoroutine(MoveCharacterToInteractable(path, target));
    }
    
    /// <summary>
    /// Moves character along path and triggers appropriate view transitions
    /// </summary>
    private IEnumerator MoveCharacterToInteractable(Vector3[] path, InteractableObject target)
    {
        // Begin character movement animation
        _activeCharacter.BeginZeroGNavigation(path);
        
        // Follow character with camera during approach
        while (_activeCharacter.IsNavigating)
        {
            // Check if we need to adjust view based on orientation
            if (ShouldAdjustViewForBetterVisibility(_activeCharacter, target))
            {
                AdjustViewForOptimalVisibility(target);
            }
            
            yield return null;
        }
        
        // Character has reached interaction position
        if (target is Doorway doorway)
        {
            // Handle doorway transition
            StartCoroutine(HandleDoorwayTransition(doorway));
        }
        else
        {
            // Handle generic interactable
            target.Interact(_activeCharacter);
            
            // Adjust view for optimal interaction visibility
            AdjustViewForInteraction(target, _activeCharacter);
        }
    }
    
    /// <summary>
    /// Manages view transition when character moves through a doorway
    /// </summary>
    private IEnumerator HandleDoorwayTransition(Doorway doorway)
    {
        // Open the door
        doorway.OpenDoor();
        yield return new WaitForSeconds(doorway.OpeningDuration);
        
        // Begin passing through doorway animation
        _activeCharacter.PassThroughDoorway(doorway);
        
        // Wait for character to begin passing through
        yield return new WaitForSeconds(0.3f);
        
        // Determine optimal view for connected room/area
        ViewConfiguration newRoomView = DetermineOptimalViewForConnectedArea(doorway);
        
        // Transition to new view
        StartCoroutine(TransitionToNewAreaView(newRoomView, _transitionDuration));
        
        // Complete doorway transition
        yield return new WaitForSeconds(0.7f);
        
        // Close door if automatic closing is enabled
        if (doorway.AutoClose)
        {
            doorway.CloseDoor();
        }
        
        // Character has completed transition
        _activeCharacter.CompleteNavigation();
    }
    
    /// <summary>
    /// Determines if current view angle is suboptimal for the interaction
    /// </summary>
    private bool ShouldAdjustViewForBetterVisibility(Character character, InteractableObject target)
    {
        // Calculate viewing angle of the interaction
        Vector3 interactionDirection = target.transform.position - character.transform.position;
        Vector3 viewDirection = _camera.forward;
        float viewingAngle = Vector3.Angle(interactionDirection, viewDirection);
        
        // Check if doorway orientation is perpendicular to current view (left/right limitation)
        if (target is Doorway doorway)
        {
            Vector3 doorForward = doorway.transform.forward;
            float doorAlignment = Mathf.Abs(Vector3.Dot(doorForward, _camera.right));
            
            // If door is aligned with camera right vector (perpendicular to view)
            if (doorAlignment > 0.7f)
            {
                return true;
            }
        }
        
        // Return true if viewing angle is suboptimal
        return viewingAngle > 60f || viewingAngle < 10f;
    }
    
    /// <summary>
    /// Adjusts view to provide optimal visibility of an interactable object
    /// </summary>
    private void AdjustViewForOptimalVisibility(InteractableObject target)
    {
        // Get target's preferred viewing angle if specified
        Quaternion preferredRotation = target.GetPreferredViewingAngle();
        
        // If no preferred angle, calculate optimal position
        if (preferredRotation == Quaternion.identity)
        {
            // For doorways, position camera to see through the door
            if (target is Doorway doorway)
            {
                // Get position slightly offset from doorway center
                Vector3 optimalPosition = doorway.transform.position + 
                    doorway.transform.up * 1.2f + 
                    doorway.transform.forward * -2f;
                
                // Interpolate camera to this position
                StartCoroutine(SmoothCameraPositionTransition(optimalPosition));
            }
            else
            {
                // For other interactables, position to see both character and object
                Vector3 midPoint = (_activeCharacter.transform.position + target.transform.position) / 2f;
                Vector3 viewOffset = Vector3.up * 1.5f + Vector3.back * 4f;
                
                StartCoroutine(SmoothCameraPositionTransition(midPoint + viewOffset));
            }
        }
        else
        {
            // Use object's preferred viewing angle
            StartCoroutine(SmoothCameraRotationTransition(preferredRotation));
        }
    }
    
    /// <summary>
    /// Transitions to optimal view for connected area when passing through doorway
    /// </summary>
    private ViewConfiguration DetermineOptimalViewForConnectedArea(Doorway doorway)
    {
        // Get connected room/area
        ShipArea connectedArea = doorway.GetConnectedArea();
        
        // Check for predefined optimal view for this area
        ViewConfiguration areaDefaultView = connectedArea.GetDefaultViewConfiguration();
        
        // If no predefined view, calculate optimal position
        if (areaDefaultView == null)
        {
            // Calculate room center
            Vector3 roomCenter = connectedArea.GetBounds().center;
            
            // Calculate entry vector (direction character is entering from)
            Vector3 entryVector = doorway.transform.forward;
            
            // Create view configuration that looks from opposite entry direction
            return new ViewConfiguration
            {
                Position = roomCenter - entryVector * connectedArea.GetViewDistance(),
                Rotation = Quaternion.LookRotation(entryVector),
                OrthographicSize = connectedArea.GetRecommendedOrthographicSize(),
                ViewMode = connectedArea.GetRecommendedViewMode()
            };
        }
        
        return areaDefaultView;
    }
    
    /// <summary>
    /// Adjusts view specifically for optimal interaction visibility
    /// </summary>
    private void AdjustViewForInteraction(InteractableObject target, Character character)
    {
        // Special case for monitors and displays
        if (target is ShipDisplay display)
        {
            // Position camera to clearly see the display screen
            Vector3 displayNormal = display.transform.forward;
            Vector3 optimalPosition = display.transform.position 
                - displayNormal * 2.0f 
                + Vector3.up * 0.5f;
            
            // Switch to appropriate view mode
            _viewManager.TransitionToView(ViewMode.Schematic2D);
            
            // Move camera to optimal position
            StartCoroutine(SmoothCameraPositionTransition(optimalPosition));
            StartCoroutine(SmoothCameraRotationTransition(
                Quaternion.LookRotation(displayNormal)
            ));
        }
    }

public abstract class InteractableObject : MonoBehaviour

    /// Returns the position where a character should move to interact

        return transform.position - transform.forward * _interactionRadius;
    }
    
    /// <summary>
    /// Called when character interacts with this object
    /// </summary>
    public virtual void Interact(Character character)
    {
        // Base implementation - override in derived classes
        Debug.Log($"Character {character.Name} interacted with {gameObject.name}");
    }
    
    /// <summary>
    /// Returns preferred camera viewing angle for this interaction, if any
    /// </summary>
    public virtual Quaternion GetPreferredViewingAngle()

        return Quaternion.identity;
    }
    
    /// <summary>
    /// Called when pointer hovers over this object
    /// </summary>
    public virtual void OnHoverEnter()
    {
        if (_highlightEffect != null)
        {
            _highlightEffect.SetActive(true);
        }
        
        _isHighlighted = true;

        UIManager.ShowInteractionPrompt(_interactionPrompt, transform.position);
    }
    
    /// <summary>
    /// Called when pointer exits this object
    /// </summary>
    public virtual void OnHoverExit()

        if (_highlightEffect != null)
        {
            _highlightEffect.SetActive(false);
        }
        
        _isHighlighted = false;
        
        // Hide interaction prompt
        UIManager.HideInteractionPrompt();
    }
}

        {
            categories.Add(new CommandCategory
            {
                Name = "Interact",
                Icon = IconLibrary.GetIcon("interact"),
                Commands = GenerateInteractionCommands(crewMember, nearbyObjects)
            });

        // Add skill-based commands (engineering, science, etc.)

            {
                Name = "Engineering",
                Icon = IconLibrary.GetIcon("engineering"),

            });

```csharp
/// <summary>
/// Example of optimized list virtualization
/// </summary>
public class VirtualizedListView : MonoBehaviour
{
    [SerializeField] private RectTransform _contentContainer;
    [SerializeField] private GameObject _itemPrefab;
    [SerializeField] private int _poolSize = 20;
    
    private List<ListItemData> _allItems = new List<ListItemData>();
    private List<ListItemView> _pooledItems = new List<ListItemView>();
    private int _firstVisibleIndex;
    
    private void Awake()
    {
        // Create object pool
        for (int i = 0; i < _poolSize; i++)
        {
            GameObject item = Instantiate(_itemPrefab, _contentContainer);
            ListItemView view = item.GetComponent<ListItemView>();
            view.gameObject.SetActive(false);
            _pooledItems.Add(view);
        }
    }
    
    public void SetItems(List<ListItemData> items)
    {
        _allItems = items;
        
        // Update content size
        float contentHeight = items.Count * ItemHeight;
        _contentContainer.sizeDelta = new Vector2(_contentContainer.sizeDelta.x, contentHeight);
        
        // Refresh visible items
        RefreshVisibleItems();
    }
    
    public void OnScroll(Vector2 scrollPosition)
    {
        RefreshVisibleItems();
    }
    
    private void RefreshVisibleItems()
    {
        // Calculate visible range
        float scrollY = _contentContainer.anchoredPosition.y;
        int startIndex = Mathf.FloorToInt(scrollY / ItemHeight);
        startIndex = Mathf.Max(0, startIndex);
        
        // Deactivate all items
        foreach (var item in _pooledItems)
        {
            item.gameObject.SetActive(false);
        }
        
        // Activate and position visible items
        int itemIndex = 0;
        for (int i = startIndex; i < startIndex + _poolSize && i < _allItems.Count; i++)
        {
            if (itemIndex < _pooledItems.Count)
            {
                ListItemView view = _pooledItems[itemIndex];
                view.gameObject.SetActive(true);
                view.SetData(_allItems[i]);
                view.SetPosition(i * ItemHeight);
                itemIndex++;
            }
        }
    }
}
```
