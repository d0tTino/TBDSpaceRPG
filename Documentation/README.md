# TBD Space Game Documentation

This directory contains the organized documentation for the TBD Space Game project. The Unity project previously stored in `TBD SpaceGame/` has been removed from the main branch and archived on the `unity-archive` branch for reference.

## Directory Structure

- **Technical**: Technical documentation, implementation details, and coding standards
  - Game systems implementation
  - Technical appendices
  - UI/UX design documents

- **Narrative**: Story, lore, and cultural evolution documentation
  - Cultural and religious evolution systems
  - Character backgrounds
  - Setting information
  - Story structure

- **Development**: Development workflow and process documentation
  - Development guides
  - Tooling setup
  - Version control guidelines
  - Workflow procedures

- **Research**: Research materials and reference documentation
  - Scientific basis for game mechanics
  - Historical references
  - Technical research
  - UI/UX research including spacecraft system interface paradigms

## File Naming Convention

Files should follow this naming pattern:
`TBD_Space_Game_[Category]_[Topic].txt`

Example: `TBD_Space_Game_Cultural_Evolution.txt`

## Version Control

All documentation files should include a version date at the top of the file.
Example: `Version: April 2025`

## Recent Updates

- June 2025: Added research on UI/UX paradigms for representing complex spacecraft systems to non-expert players, comparing diegetic, spatial 3D, traditional 2D, and hybrid approaches with quantitative metrics.
- April 2025: Added research on thermal signatures and heat dissipation for space weapons, examining implications for vessel stealth, thermal management, and tactical doctrines.
