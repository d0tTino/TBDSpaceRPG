# TBD Space Game - Development Journal

## Technical Architecture

### Core Systems
1. **MCP (Machine Control Protocol) Integration**
   - Purpose: Communication between Unity and external systems
   - Implementation: WebSocket-based communication
   - Status: Basic functionality working, optimization needed
   - Key Files:
     - mcpWebSocketClient-simple.ps1
     - MCP-PowerShell-Guide.md
     - Unity-MCP-Integration-Manual.md

2. **Ship Customization System**
   - Purpose: Allow upgrades and modifications to ships
   - Implementation: Menu-driven system via MCP
   - Status: Basic functionality working
   - Key Files:
     - ShipUpgradeMenuHelperV2.cs
     - ShipCustomizationManager.cs

3. **Time Skip Manager**
   - Purpose: Handle generational progression
   - Implementation: Complex system for aging, relationships, events
   - Status: Core functionality implemented
   - Key Files:
     - TimeSkipManager.cs
     - CrewManager.cs
     - WorldStateManager.cs
     - ChapterManager.cs

4. **Crew System**
   - Purpose: Manage crew members, relationships, skills
   - Implementation: Object-oriented system with state tracking
   - Status: Core models implemented
   - Key Files:
     - CrewMember.cs
     - Relationship.cs

## AI Collaboration Guidelines

### Context Provision
When seeking AI assistance, provide:
1. **Current focus:** What specific aspect you're working on
2. **Related files:** Which files are relevant to the task
3. **Current challenges:** What specific problems need solving
4. **Implementation constraints:** Any technical limitations

### Directory Structure Reference
- **Scripts/**: Core gameplay code
  - **Managers/**: System managers (Crew, World, Chapter, UI)
  - **Models/**: Data structures (CrewMember, Relationship)
  - **UI/**: User interface components
- **Gameplay_System/**: Game design documentation
- **Lore and Story Structure/**: Narrative documentation
- **Output/**: Compiled documentation and reports
- **MCP/**: Machine Control Protocol integration

### Common Commands
For MCP interaction:
```powershell
# Start MCP server
env UNITY_PORT=8001 node [path-to-server]/build/index.js

# Test menu item
./mcpWebSocketClient-simple.ps1 -menuPath "MCP/Ship/V2/Create Test Ship"

# Check server status
netstat -ano | findstr :8001
```
